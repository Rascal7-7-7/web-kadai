CSS

Webページのスタイルとレイアウトを定義するスタイルシート言語

基本情報

開発者
W3C
初回リリース
1996年
パラダイム
宣言型
難易度
簡単

特徴

  • HTMLと組み合わせて使用
  • セレクタとプロパティによる宣言型構文
  • カスケーディング（継承）機能
  • レスポンシブデザイン対応
  • アニメーションとトランジション
  • フレックスボックスとグリッドレイアウト
  • プリプロセッサ（Sass, Less）による拡張

主な用途

Webデザイン
レスポンシブデザイン
アニメーション
UI/UX

サンプルコード

/* 基本的なスタイル */
body {
  font-family: 'Arial', sans-serif;
  line-height: 1.6;
  color: #333;
  background-color: #f4f4f4;
  margin: 0;
  padding: 0;
}

/* コンテナ */
.container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 0 20px;
}

/* ヘッダー */
header {
  background-color: #333;
  color: #fff;
  padding: 1rem 0;
  box-shadow: 0 2px 5px rgba(0, 0, 0, 0.1);
}

header h1 {
  margin: 0;
  font-size: 1.8rem;
}

/* ナビゲーション */
nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

nav ul {
  display: flex;
  list-style: none;
  padding: 0;
}

nav ul li {
  margin-left: 20px;
}

nav a {
  color: #fff;
  text-decoration: none;
  transition: color 0.3s ease;
}

nav a:hover {
  color: #0088cc;
}

/* メインコンテンツ */
main {
  padding: 2rem 0;
}

/* カード */
.card {
  background: #fff;
  border-radius: 5px;
  box-shadow: 0 2px 5px rgba(0, 0, 0, 0.1);
  padding: 20px;
  margin-bottom: 20px;
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.card:hover {
  transform: translateY(-5px);
  box-shadow: 0 5px 15px rgba(0, 0, 0, 0.2);
}

/* ボタン */
.button {
  display: inline-block;
  background-color: #0088cc;
  color: #fff;
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  text-decoration: none;
  transition: background-color 0.3s ease;
}

.button:hover {
  background-color: #006699;
}

/* フレックスボックスレイアウト */
.flex-container {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}

.flex-item {
  flex: 1 1 300px;
}

/* グリッドレイアウト */
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
  gap: 20px;
}

/* アニメーション */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

.fade-in {
  animation: fadeIn 1s ease-in;
}

/* メディアクエリ（レスポンシブデザイン） */
@media (max-width: 768px) {
  nav {
    flex-direction: column;
  }
  
  nav ul {
    margin-top: 1rem;
  }
  
  .grid-container {
    grid-template-columns: 1fr;
  }
}

/* 変数（カスタムプロパティ） */
:root {
  --primary-color: #0088cc;
  --secondary-color: #006699;
  --text-color: #333;
  --background-color: #f4f4f4;
}

.theme-colored {
  color: var(--primary-color);
  background-color: var(--background-color);
}

/* 疑似要素 */
.quote::before {
  content: '"';
  font-size: 2em;
  color: #ccc;
}

.quote::after {
  content: '"';
  font-size: 2em;
  color: #ccc;
}

/* トランスフォーム */
.rotate:hover {
  transform: rotate(10deg);
}

.scale:hover {
  transform: scale(1.1);
}

/* トランジション */
.smooth-transition {
  transition: all 0.3s ease;
}

/* フィルター */
.blur-effect {
  filter: blur(2px);
}

.blur-effect:hover {
  filter: blur(0);
}
