HTML

Webページの構造を定義するマークアップ言語。すべてのWebサイトの基盤

基本情報

開発者
Tim Berners-Lee
初回リリース
1993年
パラダイム
マークアップ言語
難易度
簡単

特徴

  • シンプルなタグベースの構文
  • セマンティックな要素
  • アクセシビリティサポート
  • すべてのブラウザで対応
  • CSS・JavaScriptとの連携
  • レスポンシブデザイン対応
  • SEO最適化

主な用途

Webページ
Webアプリ
メール
ドキュメント

サンプルコード

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML サンプル</title>
    <style>
        body { font-family: Arial, sans-serif; margin: 20px; }
        .highlight { background-color: yellow; }
        .container { max-width: 800px; margin: 0 auto; }
    </style>
</head>
<body>
    <div class="container">
        <header>
            <h1>HTMLの基本構造</h1>
            <nav>
                <ul>
                    <li><a href="#section1">セクション1</a></li>
                    <li><a href="#section2">セクション2</a></li>
                </ul>
            </nav>
        </header>

        <main>
            <section id="section1">
                <h2>テキストと段落</h2>
                <p>これは<strong>重要な</strong>テキストです。</p>
                <p><em>強調された</em>テキストと<span class="highlight">ハイライト</span>されたテキスト。</p>
            </section>

            <section id="section2">
                <h2>リストと画像</h2>
                <ul>
                    <li>リスト項目1</li>
                    <li>リスト項目2</li>
                    <li>リスト項目3</li>
                </ul>
                
                <img src="sample.jpg" alt="サンプル画像" width="300">
            </section>

            <section>
                <h2>フォーム</h2>
                <form>
                    <label for="name">名前:</label>
                    <input type="text" id="name" name="name" required>
                    
                    <label for="email">メール:</label>
                    <input type="email" id="email" name="email" required>
                    
                    <button type="submit">送信</button>
                </form>
            </section>
        </main>

        <footer>
            <p>© 2024 HTMLサンプル</p>
        </footer>
    </div>
</body>
</html>
