Java

企業システムからモバイルアプリまで幅広く使用される、オブジェクト指向プログラミング言語

基本情報

開発者
Sun Microsystems
初回リリース
1995年
パラダイム
オブジェクト指向、命令型、ジェネリック
難易度
普通

特徴

  • 「Write Once, Run Anywhere」- 一度書けばどこでも実行可能
  • 強力なオブジェクト指向プログラミング機能
  • 自動メモリ管理（ガベージコレクション）
  • 豊富なライブラリとフレームワーク
  • マルチスレッド処理のサポート
  • セキュリティ機能が充実
  • 大規模なコミュニティとエコシステム

主な用途

Androidアプリ
Webアプリケーション
企業システム
デスクトップアプリ

サンプルコード

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
        
        // 変数の宣言と使用
        String message = "Javaプログラミング";
        int number = 42;
        
        System.out.println(message + " - " + number);
        
        // 配列の使用
        int[] numbers = {1, 2, 3, 4, 5};
        for (int num : numbers) {
            System.out.println("Number: " + num);
        }
    }
}
